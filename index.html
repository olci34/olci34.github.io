  • Space Cruise

    Code Demo

    A single page application that allows users to book trips to the planets in 3D platform. Rails, React and Redux frameworks are used. This was my first React project. My purpose was mainly getting comfortable with creating functional and class components, adapting newly introduced hooks, using Redux store efficiently and having a better understanding of React lifecycle methods. Trending space traveling news were birth place of the project idea. Also this was a great opportunity to explore Three.js library which is a cross-browser JS library that ables creating 3D interactive objects using WebGL. Playing with 3D objects, angles, vectors got me back to high school and college years. I am amazed by what Three.js can bring to frontend. Recently, I am still working on adding features such as displaying QR code tickets.

  • Icy Truck

    Code Demo

    A Rails app where users can purchase or sell ice creams from their own ice cream truck. The idea was to keep the ice cream truck culture alive. One of the challenges I faced in this project was building multiple many-to-many relationships between models. Active Record was a great tool to come over this obstacle. Also developing nested forms and model methods that respond to them was another learning experience. The app provides user authentication with Facebook using OAuth. Bcrypt is implemented to keep user password safe in database.
